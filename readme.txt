== Description ==

The Bitcoin Block Date Converter plugin allows you to convert all the dates within your WordPress posts and pages into corresponding Bitcoin block numbers. You can configure it to display the date, the block number, or both, and the plugin also includes a clickable link to view the Bitcoin block details.

With the Bitcoin Block Date Converter, you can:

- Engage crypto enthusiasts with relevant content.
- Enhance the authenticity of your crypto-related posts.
- Provide an additional layer of context related to Bitcoin's history.

The plugin is also compatible with WPML for multilanguage support.

== Installation ==

1. Upload the `bitcoin-block-date-converter` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Settings' > 'General' and configure the 'Date display option' as per your requirement.
